Introduction

This document describes the namespace http://pgxo.loria.fr/. The preferred prefix for this namespace is pgxo.

Content negotiation is supported for HTML and RDF/XML.

The OWL ontology file can be accessed directly from http://pgxo.loria.fr/data/pgxo.owl.
